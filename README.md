# Commodity-Prediction
This is a Python-based tool designed to fetch historical market data and predict future prices for commodities like Gold, Silver, and Crude Oil. It uses machine learning to provide a 30-day forecast and visualizes the results using interactive graphs.

Features ->

1. Real-time Data: Fetches the latest market prices via the yfinance API.
2. Multi-Currency Support: View forecasts in INR, USD, EUR, or GBP.
3. Accuracy Metrics: Automatically calculates the model's accuracy percentage in the terminal.
4. Visual Forecasting: Generates a trend graph comparing historical data with predicted future prices.
5. Exportable Results: Save your 30-day forecast directly to a CSV file.
   

Built With  ->

1. Python (Core Logic)
2. Scikit-Learn (Linear Regression Model)
3. Pandas & NumPy (Data Processing)
4. Matplotlib (Data Visualization)
5. yfinance (Financial Data Source)


Now, How to run this Project in your device ->

1. Clone the project in your device

   - run this command in your terminal
2. Install dependencies for this projec to run
   - pip install yfinance pandas numpy matplotlib scikit-learn
   - it will install necessary libraries to run this project
3. Run the application
   - python main.py
   - in your terminal (This will run the Project)
